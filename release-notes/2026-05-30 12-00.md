# Release 2026-05-30 — Stima SOC da MPPT: due batterie, ETA, profili separati

## Cosa è cambiato

### Due batterie, due profili
- **MPPT 1** → LiFePO4 100Ah (tensione fino a 14.4 V in assorbimento)
- **MPPT 2** → Piombo 70Ah (AGM/gel, OCV diverso)

Curve OCV separate per ciascuna chimica. La stima usa sempre MPPT 1 per la LiFePO4.

### Tabelle OCV
| LiFePO4 | Lead-acid |
|---|---|
| 14.4 V → 100% | 12.73 V → 100% |
| 13.6 V → 100% (riposo) | 12.55 V → 80% |
| 13.2 V → 75% | 12.08 V → 40% |
| 13.0 V → 35% | 11.63 V →  5% |
| Float/Storage → 100% | Float/Storage → 100% |
| Absorption → 92% | Absorption → 85% |
| Bulk → null (tensione gonfiata) | Bulk → null |

### ETA (tempo stimato)
- **Carica**: quando MPPT è in Bulk/Absorption con `battA > 1A`:  
  `ETA = Ah_rimanenti / battA_MPPT`
- **Scarico**: dal tasso di calo SOC negli ultimi 2+ minuti (minimo 5 campioni).  
  Se SOC cala < 1.5%/h il campo non viene mostrato (insufficiente per una stima affidabile).

### Dashboard — card Batterie (senza BMS)
- SOC LiFePO4 con ETA e riga secondaria Piombo (`~XX% · 12.5V`)

### Schermata Victron
- Due card separate: una per MPPT 1 (LiFePO4) e una per MPPT 2 (Piombo)
- Gauge semicircolare + badge stato + ETA per ciascuna
