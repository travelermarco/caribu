# Release 2026-05-29 — Auto-trigger biometria al lock screen

## Cosa è cambiato

### Biometria auto-avviata
Se l'impronta digitale o Face ID è abilitato, la richiesta biometrica viene ora avviata automaticamente 300 ms dopo la comparsa della schermata di blocco, senza richiedere il click manuale su "Impronta / Face ID".
Il pulsante rimane comunque visibile per un nuovo tentativo manuale in caso di fallimento.

## File modificati
- `js/lock.js` — `showLockScreen()`: estratta `tryBio()` come funzione condivisa, aggiunto `setTimeout(tryBio, 300)` dopo `appendChild`
