# Release 2026-05-28 — History, Weather, Campsites, Notifications, Energy Balance

## Cosa è cambiato

### 1. `js/history.js` — Storico dati (nuovo file)
- `pushSnapshot()`: salva snapshot ogni 5 min in localStorage (`caribu_history_v1`), throttled, auto-prune 7 giorni
- `getRecent(hours)`: recupera snapshot delle ultime N ore
- `drawHistChart()`: grafico SVG time-series con area fill gradient, y-axis labels, time labels. Mostra "Raccolta dati in corso…" se < 2 punti. ViewBox 320x90, padding {t:6,r:4,b:18,l:36}

### 2. `js/weather.js` — Meteo (nuovo file)
- `getCoords()`: geolocation con cache 10min
- `getWeather()`: fetch Open-Meteo, cache 1h, no API key required
- `wmoLabel(code)`: WMO code → emoji + testo italiano
- `solarForecast(data)`: irradianza media ore diurne oggi (W/m²)
- `renderWeather(containerId)`: card posizione, grid 3 giorni, stima produzione solare, fallback a form manuale se GPS non disponibile

### 3. `js/campsites.js` — Log campeggi (nuovo file)
- `startTracking()`: watchPosition, rileva sosta > 200m × 10min → crea campsite automaticamente
- Reverse geocoding via Nominatim (comune/città in italiano)
- Storage localStorage `caribu_camps_v1`: `{id, lat, lon, address, arrival, departure, notes}`
- `getCampsites()`, `deleteCampsite(id)`, `updateNotes(id, notes)`
- `renderCampsites(containerId)`: card con luogo, date, notti, badge "Qui ora", textarea note, tasto elimina

### 4. `js/notifications.js` — Notifiche push (nuovo file)
- `requestPermission()`: `Notification.requestPermission()`
- `checkThresholds(state)`: SOC basso/pieno, temp bassa, alert solare (pvW=0 ore 9-17 con MPPT connesso). Cooldown 30min per tipo
- `notify(title, body)`: via ServiceWorker con fallback `new Notification()`
- `getThresholds()` / `saveThresholds()`: localStorage `caribu_notif_v1`
- `renderNotifSettings(containerId)`: UI soglie + stato permesso

### 5. `index.html`
- Aggiunto `<link rel="apple-touch-icon">`
- Dashboard: card "⚡ Bilancio energetico" (hidden finché non ci sono dati) + card "📊 Ultime 24h" con 3 grafici storici
- Nuove sezioni: `#screen-meteo`, `#screen-campeggi`
- Nuovi tab: Meteo 🌤, Campeggi 📍

### 6. `js/app.js`
- Import dei 4 nuovi moduli
- Callback BMS: aggiunto `pushSnapshot`, `checkThresholds`, `renderHistCharts`, `renderEnergy`
- Callback MPPT: aggiunto `renderEnergy`
- `renderEnergy()`: bilancio solare → batteria → carichi con barra proporzionale
- `renderHistCharts()`: disegna i 3 grafici storici nella dashboard
- `renderMeteo()` / `renderCampeggi()`: render dei nuovi tab
- `renderSettings()`: aggiunto card notifiche con `renderNotifSettings`
- Tab routing: gestione click su meteo/campeggi
- `init`: aggiunto `startTracking()` e `renderHistCharts()`
- Globals: `refreshCampeggi`, `deleteCamp`, `updateCampNotes`, `requestNotifPerm`, `saveNotifThreshold`

### 7. `styles.css`
- Tab bar scrollabile (overflow-x: auto, scrollbar hidden, tab min-width 58px)
- `.energy-flow`, `.energy-node`, `.energy-icon`, `.energy-val`, `.energy-unit`, `.energy-lbl`, `.energy-arrow`
- `.hist-empty`
- `.weather-grid`, `.weather-day`, `.weather-day-name`, `.weather-emoji`, `.weather-temps`, `.weather-desc`, `.weather-rain`
