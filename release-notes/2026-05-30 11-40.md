# Release 2026-05-30 — Stima SOC batteria da MPPT

## Cosa è cambiato

### Stima percentuale batteria da tensione Victron MPPT
Il regolatore di carica (MPPT) legge la tensione batteria ad ogni advertisment BLE.
Quella tensione viene ora convertita in una stima di SOC tramite lookup table LiFePO4 12V.

**Logica `voltageToSOC(v, csNum)`:**
| Stato di carica | SOC restituito |
|---|---|
| Float (5) / Storage (6) | 100% — batteria piena |
| Absorption (4) | ~92% — quasi piena |
| Bulk (3) | null — tensione gonfiata, non stimabile |
| Off / altro | lookup tabella OCV (13.60 V → 100%, 12.00 V → 0%) |

### Dashboard — card Batterie
Se il BMS Bluetooth non è connesso ma l'MPPT lo è, la card mostra:
- `~XX%` (stima) + tensione + stato di carica
- "In carica" se in fase Bulk
- Label "stima MPPT · LiFePO4"

### Schermata Victron — card "Batteria"
Quando almeno un MPPT è connesso, compare una card con:
- Gauge semicircolare identico alla schermata BMS (`~XX%`)
- Badge stato di carica + card tensione
- Nota precisione ±5%
